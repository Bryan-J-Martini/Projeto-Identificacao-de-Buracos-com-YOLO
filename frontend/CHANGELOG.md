Este arquivo explica como Visual Studio criou o projeto.

As seguintes ferramentas foram usadas para gerar este projeto:
- create-vite

As etapas a seguir foram usadas para gerar este projeto:
- Crie um projeto react com create-vite: `npm init --yes vite@latest front_projeto_buracos -- --template=react  --no-rolldown --no-immediate`.
- Atualizando `vite.config.js`com a porta.
- Criar o arquivo de projeto (`front_projeto_buracos.esproj`).
- Crie `launch.json` para habilitar a depuração.
- Adicionar projeto à solução.
- Grave este arquivo.
